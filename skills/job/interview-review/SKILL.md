> context: 프론트엔드 개발자. TypeScript 베이스. React, Vue 경험.

## 평가 척도

답변을 아래 5개 항목으로 평가합니다. ★ 5점 만점 (0.5 단위).

| 항목        | 설명                    | ★★★★★ 기준                 |
|-----------|-----------------------|--------------------------|
| **정확성**   | 기술적으로 틀린 내용이 없는가      | 오류 없이 정확한 개념 전달          |
| **깊이**    | 표면적 답변을 넘어 원리까지 설명하는가 | 내부 동작, 트레이드오프, 히스토리까지 언급 |
| **구조화**   | 답변이 논리적으로 정리되어 있는가    | 두괄식, 단계적 설명, 결론 명확       |
| **실무 연결** | 실제 경험이나 사례와 연결하는가     | 구체적 프로젝트/상황에서의 적용 사례 포함  |
| **간결성**   | 군더더기 없이 핵심을 전달하는가     | 1~2분 내 답변 가능한 밀도         |

### 등급 기준

| 점수           | 의미                   |
|--------------|----------------------|
| ★★★★★        | 시니어 수준. 면접관이 감탄할 답변  |
| ★★★★ ~ ★★★★½ | 좋음. 합격권 답변           |
| ★★★ ~ ★★★½   | 보통. 틀리진 않지만 인상적이지 않음 |
| ★★ ~ ★★½     | 부족. 핵심을 놓치거나 부정확     |
| ★ ~ ★½       | 심각. 오개념이거나 답변 불가 수준  |
